== Description ==

Setup your site theme from [WEB Theme Space](https://www.wpbusinessthemes.com/) site with template library dummy data easily. Import settings, widgets and content with one click. TSP Demos Setup requires [Advanced Import](https://wordpress.org/plugins/advanced-import/) Plugin to work normally.

While you use TSP Demos Setup to import demo starter site, Images and demo files are fetches from respected theme Demo Sites form WEB Theme Space. This helps you to import starter dmeo site with a single click. You must accept [terms](https://www.wpbusinessthemes.com/terms-and-conditions/) and [privacy](https://www.wpbusinessthemes.com/privacy-policy/) to use TSP Demos Setup Plugin.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'tsp-demos'
3. Activate TSP Demos Setup from Appearance > Plugins
4. Go to the Appearance -> Demo Import
5. Use available demo import

= From WordPress.org =

1. Download tsp-demos.zip
2. Upload the 'tsp-demos' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate tsp-demos from your Appearance > Plugins page.
4. Go to the Appearance -> Demo Import
5. Use available demo import
